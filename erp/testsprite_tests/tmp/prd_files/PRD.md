# PT Indo OceanCrew Services - ERP System
## Product Requirements Document

## Overview
A comprehensive ERP web application for PT Indo OceanCrew Services, a professional crewing agency. Manages crew operations, contracts, payroll, finance, recruitment, vessels, and documents.

## Modules
1. Authentication - Login, password reset, OTP
2. Dashboard - Statistics, charts, quick actions
3. Crew Management - Profiles, documents, status tracking
4. Contract Management - Create, approve, renew, terminate
5. Vessel Management - Register vessels, assign crew
6. Payroll - Salary calculation, payslips
7. Finance - Chart of accounts, journal entries, invoices, bills
8. Recruitment Hub - Pipeline, referral codes, recruiter performance
9. Document Management - Upload, categorize, expiry tracking
10. Analytics & Reporting - Crew analytics, financial reports
11. Monitoring - Integration monitoring, activity tracking
12. Settings - System config, user management, notifications
13. Smart Import - Excel/CSV import with validation
14. Client Management - Client profiles and agreements
